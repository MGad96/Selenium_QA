1) Download VS Code	
2) Download DOT Net Core SDK 
3) Make sure you have the following VScode extensions 
	- Nuget Package Manager
	- C# language 
	- .NET Core Test Explorer 

4) Install Selenium Webdriver 
5) Ensure chromedriver.exe is in the right folder project/bin/debug/net5.0
6) Run the login tests and dynamic controls tests through the .NET Core Test Explorer 
7) Run the tables sorting through normal terminal run (F5)
